\par
\item Training Set adalah set digunakan oleh algoritma klassifikasi . Dapat dicontohkan dengan :  decision tree, bayesian, neural network dll. Semuanya dapat digunakan untuk membentuk sebuah model classifier.

\end{enumerate}
